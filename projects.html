Imperative Language Compiler

This project focuses on the creation of a compiler for a simple imperative programming language. It is designed to convert source code into virtual machine instructions, supporting basic constructs such as variables, arrays, arithmetic, loops, conditionals and procedures. Built using Python and the SLY library, the compiler performs both lexical and syntactic analysis, ensuring that the source code adheres to the language's rules. In addition, it identifies semantic errors and generates optimized machine code, which is then executed on a virtual machine.

Strategic Go Game with Client-Server Architecture

This project is a strategic Go game implemented with a client-server architecture, where the server manages the game logic and maintains a SQL database to store game data. The backend is written in Java, providing a scalable and efficient environment for real-time multiplayer gameplay. The client offers an intuitive interface and supports seamless interaction with the server. The game ensures real-time synchronization of game state across multiple clients, making it suitable for multiplayer gaming experiences. 

                
                public int calculateBreaths(StonesGroup group){
                    int[][] fields = {{0,-1},{0,1},{-1,0},{1,0}};
                    int breaths = 0;
                    List<String> neighboringFields = new ArrayList<>();
                    for(SingleStone stone : group.getStones()){
                        for(int[] dir: fields) {
                            int currX = stone.getX() + dir[0];
                            int currY = stone.getY() + dir[1];
                            String field = currX+" "+currY;
                            if(isEmpty(currX, currY) && !neighboringFields.contains(field)){
                                breaths += 1;
                                neighboringFields.add(field);
                            }
                        }
                    }
                    return breaths;

Key features:

  • Intuitive client interface
  • Real-time multiplayer support
  • Game state management with SQL database
  • Client-server communication with data synchronization
