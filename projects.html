Imperative Language Compiler

This project focuses on the creation of a compiler for a simple imperative programming language. It is designed to convert source code into virtual machine instructions, supporting basic constructs such as variables, loops, and conditionals. Built using Python and the SLY library, the compiler performs both lexical and syntactic analysis, ensuring that the source code adheres to the language's rules. In addition, it identifies semantic errors and generates optimized machine code, which is then executed on a virtual machine.
